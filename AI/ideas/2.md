## 1. Motivation

Linear attention 中的状态是递归累积的。若 rollout 过程中出现错误 token，这些错误会被写入内部状态，并持续影响后续计算。于是误差不仅体现在当前 token 上，还会通过 state 传播，削弱后续位置上的自我纠正能力。

因此，希望在 midtraining 阶段显式训练这样一种能力：即使前缀已经被错误 rollout 污染，后续预测仍尽量接近“前缀正确时”的行为，同时内部 state 也尽量不被错误前缀带偏。

目标可概括为：提升 linear attention 在 on-policy rollout 偏差下的 self-correction 能力，并减少错误 token 对 state 的持续污染。

这里采用方案 B：分别在错误前缀与正确前缀条件下各自继续 rollout，并让错误前缀路径上的行为向正确前缀路径上的行为对齐。这样可以避免构造不自然的混合前缀，同时直接在模型自身 rollout 分布上进行优化。

---

## 2. 符号定义

设 $x$ 为输入上下文，$y$ 为 ground-truth continuation 中选取的一段真实片段，$\hat y$ 为模型在条件 $x$ 下 rollout 得到的一段预测片段，$\hat z$ 和 $z$ 分别为在两种前缀条件下继续 rollout 得到的后续序列。

即

$$\hat y \sim p_\theta(\cdot \mid x),$$

$$\hat z \sim p_\theta(\cdot \mid x,\hat y),$$

$$z \sim p_\theta(\cdot \mid x,y).$$

对后续序列中的位置 $t$，考虑两种条件分布：

错误前缀路径上的分布为

$$p_\theta(\hat z_t \mid x,\hat y,\hat z_{<t}).$$

正确前缀路径上的分布为

$$p_\theta(z_t \mid x,y,z_{<t}).$$

记 linear attention 的内部状态为 $\mathrm{state}_t(x,\hat y,\hat z_{<t})$ 和 $\mathrm{state}_t(x,y,z_{<t})$。若只关注累计状态，则记为 $S_t(x,\hat y,\hat z_{<t})$ 和 $S_t(x,y,z_{<t})$。

---

## 3. 训练流程

### Step 1: 构造错误前缀

给定输入 $x$，先由模型 rollout 出一段预测前缀 $\hat y$，其对应的真实片段为 $y$。可写为

$$\hat y \sim p_\theta(\cdot \mid x).$$

这一步显式构造模型自身的 on-policy 偏差，而不是始终只在 teacher-forcing 条件下训练。

### Step 2: 在两种前缀下分别继续 rollout

随后分别在两种前缀条件下继续 rollout：

错误前缀路径为

$$\hat z \sim p_\theta(\cdot \mid x,\hat y),$$

正确前缀路径为

$$z \sim p_\theta(\cdot \mid x,y).$$

实现上，$\hat y,\hat z$ 与 $y,z$ 可分别从对应前缀出发连续 rollout 得到。

### Step 3: 构造 clean-vs-corrupted 对比监督

同时比较：

* 错误前缀路径：$x,\hat y,\hat z$
* 正确前缀路径：$x,y,z$

然后在每个位置 $t$ 上，让错误前缀路径上的后续行为向正确前缀路径上的行为靠近。

这里的关键在于：两条路径均在各自前缀条件下继续 rollout，而不共享后续 token，从而保持轨迹的自然性，并直接对齐两种前缀诱导出的行为差异。

### Step 4: 对齐内部状态

除输出分布外，再约束内部状态满足 $\mathrm{state}_t(x,\hat y,\hat z_{<t}) \approx \mathrm{state}_t(x,y,z_{<t})$，以直接减轻错误前缀对 memory/state 的污染。

---

## 4. Loss 设计

总目标由主 loss 和一个 state 对齐 loss 组成。主 loss 有两种候选形式。

---

## 4.1 主 loss 方案：KL / distillation

(仿照ON-POLICY DISTILLATION OF LANGUAGE MODELS:
LEARNING FROM SELF-GENERATED MISTAKES)

第一种做法是直接对齐两种路径上的输出分布。对后续位置求和，有

$$\mathcal{L}*{\mathrm{KL}} = \sum_t \mathrm{KL}\left( p_\theta(\cdot \mid x,\hat y,\hat z_{<t})  || sg(p_\theta(\cdot \mid x,y,z_{<t}) ) \right).$$

这可以看作一种 on-policy distillation：错误前缀路径上的模型行为作为 student，正确前缀路径上的模型行为作为 teacher。作用是直接在输出分布层面实现纠偏。

---

## 4.3 State 对齐 loss

除主 loss 外，再加入一个针对内部状态的对齐项。

对后续位置 $t$，设

* $a = S_t(x,\hat y,\hat z_{<t})$ 为错误前缀路径上的状态，
* $b = S_t(x,y,z_{<t})$ 为正确前缀路径上的状态，

其中 $b$ 采用 stop-gradient（即不对其反向传播梯度）。

定义逐步权重为

$$w = \left(\frac{t+1}{T}\right)^2.$$

同时定义

$$\text{cos\_loss} = 1 - \mathrm{cosine_similarity}(a,b),$$

$$\text{norm\_loss} = \left( |a| - |b| \right)^2.$$

则每个位置 $t$ 的 state 对齐 loss 为

$$\mathcal{L}_{\mathrm{state},t} = w \cdot \left( \text{cos\_loss} + 0.01 \cdot \text{norm\_loss} \right).$$

该 loss 在所有 layer 的所有 state tensor 上取平均，并在时间维度上求和，得到最终的 state 对齐项

$$\mathcal{L}_{S} = \sum_t \mathcal{L}_{\mathrm{state},t}.$$

该项的作用是不仅让 logits 接近，也让错误前缀路径诱导出的内部 memory 在方向与尺度上逐步接近正确前缀路径下的 memory，从而减少 state contamination，同时避免对表示施加过强的逐点约束。


---

## 5. 总目标

若主 loss 采用 KL 版本，则总目标为

$$\mathcal{L} = \mathcal{L}_{\mathrm{KL}} + \lambda_{\mathrm{state}} \mathcal{L}_{\mathrm{state}}.$$

若主 loss 采用 PPO / PG 版本，则总目标为

$$\mathcal{L} = \mathcal{L}_{\mathrm{PPO}} + \lambda_{\mathrm{state}} \mathcal{L}_{\mathrm{state}}.$$

---

## 6. 总结

整个方案的核心是在 midtraining 中显式构造 on-policy 错误前缀 $\hat y$，并分别在错误前缀与正确前缀条件下继续 rollout，得到两条各自的后续轨迹 $\hat z$ 与 $z$，然后让错误前缀路径上的行为向正确前缀路径上的行为对齐。

其中，主 loss 可以取：

* 基于分布匹配的 KL / distillation 目标；
* 基于 importance sampling 的 PPO-style 纠偏目标。

此外，再加入 state-level MSE loss，对齐 linear attention 的内部状态 $S$。这样既约束输出行为，也直接约束 memory/state，从而增强 rollout 误差下的自我纠正能力，并减轻错误 token 对状态的长期污染。
